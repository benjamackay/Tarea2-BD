Primero checkear que la dirección de la base de datos en el .env se encuentre correctamente seteada junto a la de el puerto default de elysia (3000)

Luego correr TablasBD.py para poblar la base de datos con las tablas vacías. (Se asume que hay mínimo un usuario registrado en la base de datos)

Por ultimo iniciar el servidor elysia tener funcionando nuestra base de datos y correr cliente.py en la carpeta cliente.
